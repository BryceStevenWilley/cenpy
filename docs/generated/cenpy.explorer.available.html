cenpy.explorer.available¶

cenpy.explorer.available(verbose=True)[source]¶

Returns available identifiers for Census Data APIs. NOTE: we do not support the Economic Indicators Time Series API yet.

Parameters
verbosebool

whether to provide ID and title or just ID (default: True)

Returns
list

identifiers (if verbose: and dataset names)
